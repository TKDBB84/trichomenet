This sensitivity is a derivation from ImageJ's 'Find Maxima' function's Noise Tolerance argument, as implemented by FIJI. Higher numbers will mark more trichomes, lower numbers will mark fewer trichomes.

More information about the 'Find Maxima' function can be found in section 29.4 of ImageJ's user's guide here: http://rsbweb.nih.gov/ij/docs/guide/146-29.html
